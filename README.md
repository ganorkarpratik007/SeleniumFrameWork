# Selenium Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Apache POI (Data-driven testing)
- Jenkins (CI-ready)

## Framework Overview
This project demonstrates a scalable Selenium automation framework using TestNG and Maven.
It follows Page Object Model principles and supports data-driven testing and reporting.

## Project Structure
- src/main/java – Framework core (Base classes, utilities)
- src/test/java – Test cases and page objects
- pom.xml – Dependency and build management
